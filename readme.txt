Cansado de usar o Wiki Media? Dias e dias lutando com o Wordpress para que seu projeto fique OK?

Então se tudo isto é verdade, o Plugin Wordpress Wiki é a reposta para seu problema! O novo Plugin Wordpress Wiki foi feito pelos “caras” da Instict (os mesmos criadores do Plugin para comércio eletrônico <a title="wordpress e-commerce plugin" href="http://www.instinct.co.nz/e-commerce">Wordpress e-Commerce Plugin</a>) adicionando a função de Wiki ao seu site baseado em Wordpress.

== Installation ==

1. Copie os arquivos do plugin para a pasta <code>wp-content/plugins/<code>

2. Adicionando a função de Wiki ao seu site baseado em Wordpress.
